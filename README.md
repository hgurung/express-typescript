## Description

[Express](https://expressjs.com/) framework boilerplate in typescript.


## Installation

```bash
$ npm install
```

## Running the app + migrate + seed

```bash
# development
$ npm run start

# build and compile
$ npm run build

# run migration
$ npm run migrate

# run seed
$ npm run seed
```

## Test

```bash
# unit tests
$ npm run test
```
